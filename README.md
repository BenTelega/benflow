# Benflow - Моя конфигурация Oh My Zsh

Этот проект содержит мою переносимую конфигурацию Oh My Zsh, которую можно легко развернуть на любой новой машине.

## Быстрый старт

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/ваш-профиль/benflow.git ~/dev/benflow
   ```

2. **Запустите скрипт установки**:
   ```bash
   cd ~/dev/benflow
   bash setup_ohmyzsh.sh
   ```

3. **Примените изменения**:
   ```bash
   source ~/.zshrc
   ```

Готово! Ваша конфигурация Oh My Zsh настроена.

## Что включено

- **Oh My Zsh**: Полная конфигурация с популярными плагинами
- **Powerlevel10k**: Красивая и функциональная тема
- **Плагины**: zsh-autosuggestions, zsh-syntax-highlighting
- **Кастомные настройки**: Мои личные алиасы и функции

## Структура проекта

```
benflow/
├── configs/
│   └── oh-my-zsh/          # Конфигурация Oh My Zsh
│       ├── .zshrc          # Основной конфиг
│       ├── .p10k.zsh       # Конфиг Powerlevel10k
│       ├── custom/         # Кастомные плагины и темы
│       └── oh-my-zsh/      # Основные файлы Oh My Zsh
├── setup_ohmyzsh.sh        # Скрипт установки
└── README.md              # Эта инструкция
```

## Как обновлять

Просто выполните `git pull` в папке проекта, и изменения автоматически применятся благодаря симлинкам.
