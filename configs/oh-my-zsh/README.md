# Настройка переносимой конфигурации Oh My Zsh

## Что было сделано

1. **Создана папка для конфигурации**: `~/dev/benflow/configs/oh-my-zsh/`
2. **Перенесены все конфиги**: `.zshrc`, `.p10k.zsh`, кастомные плагины и темы
3. **Настроены симлинки**: Все конфиги теперь ссылаются на новую папку в проекте
4. **Создан скрипт установки**: `setup_ohmyzsh.sh` для развертывания на новой машине

## Структура папки

```
~/dev/benflow/configs/oh-my-zsh/
├── .zshrc                  # Основной конфиг Zsh
├── .p10k.zsh               # Конфиг Powerlevel10k
├── custom/                 # Кастомные плагины и темы
│   ├── plugins/            # Кастомные плагины
│   ├── themes/             # Кастомные темы
│   └── completions/        # Кастомные дополнения
├── oh-my-zsh/             # Основные файлы Oh My Zsh
├── setup_ohmyzsh.sh        # Скрипт для установки на новой машине
└── README.md              # Инструкции
```

## Как использовать на новой машине

1. **Скопируйте весь проект** на новую машину:
   ```bash
   git clone ваш-репозиторий ~/dev/benflow
   ```
   или скопируйте только папку конфигурации:
   ```bash
   scp -r ~/dev/benflow/configs пользователь@новая-машина:~/dev/benflow/
   ```

2. **Запустите скрипт установки**:
   ```bash
   cd ~/dev/benflow
   bash setup_ohmyzsh.sh
   ```

3. **Примените изменения**:
   ```bash
   source ~/.zshrc
   ```

4. **Настройте Powerlevel10k** (опционально):
   ```bash
   p10k configure
   ```

## Важные заметки

- **Симлинки**: Все конфиги в домашней директории (`~/.zshrc`, `~/.p10k.zsh`, `~/.oh-my-zsh`) являются симлинками на файлы в `~/dev/benflow/configs/oh-my-zsh/`
- **Обновления**: При обновлении Oh My Zsh, обновляйте файлы в `~/dev/benflow/configs/oh-my-zsh/oh-my-zsh/`
- **Резервные копии**: Оригинальный Oh My Zsh сохранен в `~/.oh-my-zsh-backup`

## Как добавлять новые плагины

1. Установите плагин в кастомную папку:
   ```bash
   git clone https://github.com/имя/плагин.git ~/dev/benflow/configs/oh-my-zsh/custom/plugins/плагин
   ```

2. Добавьте плагин в `.zshrc`:
   ```bash
   plugins=(... новый-плагин)
   ```

3. Примените изменения:
   ```bash
   source ~/.zshrc
   ```

## Восстановление оригинальной конфигурации

Если нужно вернуть оригинальную конфигурацию:
```bash
mv ~/.oh-my-zsh-backup ~/.oh-my-zsh
mv ~/.zshrc.backup* ~/.zshrc
mv ~/.p10k.zsh.backup* ~/.p10k.zsh
```
